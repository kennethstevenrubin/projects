Example

Specify valid values for any of the fields below to modify the behavior of the Movie control.
Then press the "Create movie" button to create an instance.

Note: Clicking anywhere dismisses the movie once it is running.

title


Title of dialog. Displays at the top of the dialog and in a slightly larger font.

paragraphs




Type the paragraphs displayed as the contents of the dialog.

button caption


The caption of the "dismiss" button.

top (pixels, 0..1000)


The y-coordinate of the top of the Movie canvas.

overlay opacity (0..1)


The darkness of the overlay--indicating that the background is blocked.

back color (r,g,b)


The color of the background.

text color (r,g,b)


The color of the text.

required width (pixels, 50..1000)


The width of the movie.

required height (pixels, 50..1000)


The height of the movie.

font family


The font family for the text.

font size (pixels, 1..100)


The font size for the text.

title font size (pixels, 1..100)


The font size for the title.

button font size (pixels, 1..100)


The font size for the button.

line height (pixels, 1..100)


The number of pixels separating successive rows of text.

paragraph spacing gap (pixels, 1..100)


The number of pixels separating successive paragraphs of text.

corner radius (pixels, 0..100)


The roundness of the corners of the background and also the border spacing for the paragraphs and button.

right gap (pixels, 0..500)


The spacing between the left of the button and the right size of the dialog.

bottom gap (pixels, 0..500)


The spacing between the top of the button and the top of the bottom border of the dialog.

button height (pixels, 1..100)


The height of the button (in pixels).

title x (pixels, 1..100)


The x-coordinate of the title left.

title y (pixels, 1..100)


The y-coordinate of the title top.

paragraph start y (pixels, 1..1000)


The starting y-coordinate of the paragraphs.

width paragraph padding (pixels, 1..100)


The padding for the line-splitting paragraph operations.

line start x (pixels, 1..100)


he starting x-coordinate of the paragraph text output.
